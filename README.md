# countbook

## Description

A simple one-page site to record different dayly statistics, e.g. product sales, performed operations at work, etc.

## Technology stack

- Debian
- Nginx
- Supervisord
- Daphne
- Django + IPC channel backend
- SQLite
- Vue.js + Bulma + Font Awesome + Chart.js
